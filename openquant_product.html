产品概述与架构

OpenQuant 是面向量化交易者的高频交易 SaaS 平台，提供从策略开发、回测验证、实盘部署、托管运行到策略共享租赁的全链路一站式服务。核心引擎采用 Rust 构建，系统延迟 <10μs，支持 Binance / OKX / Gate / Bybit 等主流交易所，兼容 Python / Rust 双语言策略。

🎯 面向策略开发者

策略上传 → 代码编辑 → 回测验证 → 实盘部署 → 租赁变现，全链路闭环

👥 面向普通交易者

围观优质实盘 → 租用经验证策略 → 一键运行，低门槛参与量化

🖥️ 面向托管服务提供者

部署托管者节点，为平台用户提供服务器资源

🌐 社区生态

论坛、围观、排行榜构成社区飞轮，持续吸引量化交易者

链上交易支持、AI 辅助策略、机构私有化部署、策略众筹资金池、移动端 App

首页 · 核心区展示产品核心卖点，背景为实时行情数字矩阵

控制台

✅ 已上线
用户登录后的核心枢纽页面。包含数据概览（实盘数/策略数/交易所/托管者）、10 个快速入口、历史搜索，以及右侧用户信息面板和消息通知列表。
控制台 · 数据概览 + 快速入口 + 右侧用户面板与消息列表
⚡

实盘管理

✅ 已上线
统一管理所有交易实盘的生命周期。提供实盘总数/运行中/已停止三项统计，支持关键词+状态筛选，批量启动/停止/重启/删除，以及财务汇总（总余额、总利润等）。
实盘管理 · 统计卡片 + 筛选条件 + 批量操作工具栏

状态管理

支持 运行中 / 暂停 / 已停止 / 异常 四种状态，异常时自动重启（最多3次）

财务汇总

列表上方展示总余额、总利润、当日余额变化、当日利润变化聚合数据

⚡
优化建议
  • 列表行可点击进入"实盘监控详情页"（待开发），查看实时收益曲线 + 持仓 + 风控
  • 增加实盘模板功能：保存常用配置（交易所 + 策略组合），快速创建新实盘
  • 批量操作时增加操作日志预览（显示将要影响的实盘列表确认弹窗）

策略库（私有）

✅ 已上线
管理用户上传的私有量化策略文件，支持出租（上架租赁市场）、分享（生成链接或发布围观广场）、编辑和删除。私有策略与已发布租赁策略通过 Tab 切换。
策略库 · 展示策略文件大小、版本、运行实盘数等关键信息

在线代码编辑器

✅ 已上线
专业的在线代码编辑器，支持 Python / Rust 策略开发。具备语法高亮、实时错误检查、多标签页编辑，以及与 VS Code 插件的联动支持。

智能补全

支持 Python 内置函数、OpenQuant API（context. / data.）、pandas / numpy 方法自动补全

实时错误检查

语法错误（红色波浪线）、未定义变量、缩进错误等实时提示，底部状态栏显示错误数量

多标签页

同时打开多个策略文件（免费版最多3个，VIP 无限），未保存时显示 * 标记

搜索替换

Ctrl+F 搜索（支持正则）、Ctrl+H 替换、Ctrl+Shift+F 跨文件搜索

⚡
优化建议
  • 策略模板库：内置"双均线 / 网格 / RSI 反转"等常用策略模板，一键加载到编辑器，降低上手门槛（尤其适合实战营学员）
  • 右侧文档面板：编辑器侧边栏展示 OpenQuant API 文档，避免切出编辑器查文档
  • 代码一键回测：编辑器顶部增加"▶ 快速回测"按钮，无需离开页面即可触发回测任务
  • 长期规划：接入 AI 代码助手（类 GitHub Copilot），输入注释自动生成策略代码片段

托管者

✅ 已上线
管理运行实盘的服务器节点
托管者 · 服务器卡片展示资源使用进度条，回报时间及状态标签

交易所管理

✅ 已上线
统一管理各交易所 API 凭证。支持 Binance / OKX / Gate / Bybit 等主流交易所，Spot 和 Swap 产品类型。API Key 部分展示，绑定时自动进行权限检查（含提现权限的密钥进行提醒）。
交易所管理 · API凭证列表，支持多个交易所多账户管理

租赁管理

✅ 已上线
策略开发者发布和管理可租赁策略的页面。设置租赁策略名称、定价类型（按天/月/年）和周期。平台抽成 30%，开发者得 70%，每月 15 日自动结算。
租赁管理 · 展示租赁策略编号、密钥、绑定私有策略与定价信息

分享管理

✅ 已上线
统一管理用户对外分享的实盘、策略、交易所记录。支持两种访问权限：匿名访问（任何人通过链接访问）和公开访问（上架围观广场，全平台可见）。
分享管理 · 记录分享编号、标题、访问量、主体类型等信息

订单管理

✅ 已上线
记录用户在平台产生的所有消费订单（主要为策略租赁费用）。展示订单金额、状态、资源定价类型、订阅周期等完整信息，支持按订单号、状态、资源类型筛选。
订单管理 · 多字段筛选 + 完整订单信息表格

账号设置

✅ 已上线
三 Tab 结构：账户信息（个人资料、会员信息、通知设置）、API 接口管理（平台开放 API 密钥管理）、账户安全（密码修改、登录设备管理、2FA）。
账号设置 · 个人信息、会员状态、联系方式、安全设置

围观（公开实盘）

✅ 已上线
向所有用户（含游客）公开展示平台上优质运行实盘。卡片式网格布局（3列），展示收益曲线、累计收益、收益率、账户余额，官方认证标签提升信任度。支持按交易所筛选和综合/最新/最热/收益最高排序。
围观 · 实时追踪策略运行动态，官方认证实盘卡片展示
⚡
优化建议
  • 卡片增加迷你收益曲线（7 日迷你折线），让用户在不点击详情的情况下快速感知策略稳定性
  • 增加对比视图：同时选中 2-4 个策略进行收益 / 回撤 / 胜率横向对比
  • 与"策略租赁市场"打通：围观页直接展示该策略是否可租，以及当前租金

社区论坛

✅ 已上线
量化交易者知识与经验分享社区。8个板块分类（源码解析、经验交流、策略分享、知识库等），支持帖子发布（富文本 + 代码块 + 图片 + 回测报告插入），最新/最热排序，用户等级体系（Lv.0～Lv.5）。
社区 · 量化交易者聚集地，分享策略、探讨技术、结识同好
⚡
优化建议
  • 帖子编辑器增加"插入回测报告"组件：一键嵌入图表卡片，让分享更直观可信
  • 增加"因子广场"板块入口（对应运营计划 2.3），形成社区内的因子提交流量
  • 用户主页增加策略展示区块（已发布策略（围观） / 已上架租赁（租赁）），形成个人量化名片
  • 为21天实战营学员设置专属身份标识，鼓励在社区分享学习成果（互相背书）

策略租赁市场

✅ 已上线
策略的公开商城，连接策略开发者（卖方）与普通交易者（买方）。支持按语言（Python/Rust）、类别（加密货币/模版/插件）和技术指标（ATR/EMA/MA/RSI/SMA）双级筛选，以及关键词搜索。
策略租赁市场 · 双级分类筛选，支持按技术指标快速定位策略
